# Solitaire
A simple Solitaire card game made in Unity 2021 LTS using Model-View-Presenter architecture, dependency injection, reactive and asynchronous programming.

**Unity packages:**
* Unity UI
* TextMeshPro
* Test Framework
* Advertisement

**3rd party plugins:**
* Zenject / Extenject
* UniRx
* UniTask
* DOTween
* NSubstitute
